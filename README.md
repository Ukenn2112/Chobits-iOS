# Bangumi

[![](https://dcbadge.limes.pink/api/server/nZPTwzXxAX?style=flat)](https://discord.gg/nZPTwzXxAX)

一个基于 SwiftData & SwiftUI 的 Bangumi iOS 客户端。

完全使用 [Private API](https://github.com/bangumi/server-private) 获取数据。

原生并尽可能贴近网页版体验的 UI。
